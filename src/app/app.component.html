SCSS architecture boilerplate for Angular

      Which changes should I made in in angular.json?

        copy src/styles folder in your project:


        do below configuration:
        
          architect>build>options>
          "stylePreprocessorOptions":
            "includePaths": ["src/styles"]

        
        and
        
          architect>build>options>
          "styles":[
          "pure-start/pure-start.css", // Add this line
          "src/styles/global/styles.scss", // Add this line
          "path/to/other/styles.css"
        ],
